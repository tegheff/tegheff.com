          Project 1

Project Name

SDN Crime Analytic ML Model


Project Type

Software Engineering, Machine Learning, Flask Development, Python, HTML.


About

Utilized publicly available criminal data, released by the City of Los Angeles, to build a machine learning model capable of predicting the type of crime being committed based on diverse factors such as time, day, age, sex, location, and more. This data includes every single crime committed from 2020 to 2025.


With over 1 million entries, it would be physically impossible to compile and compare the relationships between this data by hand, which is why I built a model from scratch to do it for me.


Purpose

To explore and model relationships between perp data and their respective crime committed. Allowed me to explore machine-learning possibilities in a way I found interesting.

Codespace

# Code written by Tegan Heffernan, Data provided by the City of Los Angeles. import pandas as pd import joblib from sklearn.model_selection import train_test_split from sklearn.tree import DecisionTreeClassifier from sklearn.metrics import classification_report data = pd.read_csv('Crime_Data_from_2020_to_Present.csv').head(800000) top_codes = data['Crm Cd'].value_counts().head(10).index filtered_data = data[data['Crm Cd'].isin(top_codes)].copy() # Extract date features again from filtered data filtered_data['DATE OCC'] = pd.to_datetime(filtered_data['DATE OCC'], format='%m/%d/%Y %I:%M:%S %p', errors='coerce') filtered_data['DayOfWeek'] = filtered_data['DATE OCC'].dt.dayofweek features = filtered_data[['DayOfWeek', 'TIME OCC', 'AREA', 'Vict Age', 'Vict Sex', 'Vict Descent', 'LAT', 'LON']] target = filtered_data['Crm Cd'] from sklearn.preprocessing import LabelEncoder for col in ['Vict Sex', 'Vict Descent']: features.loc[:, col] = LabelEncoder().fit_transform(features[col]) from sklearn.model_selection import train_test_split X_train, X_test, y_train, y_test = train_test_split(features, target, test_size=0.2, random_state=42) from sklearn.tree import DecisionTreeClassifier model = DecisionTreeClassifier(random_state=42) model.fit(X_train, y_train) from sklearn.metrics import classification_report predictions = model.predict(X_test) print(classification_report(y_test, predictions, zero_division=0)) vict_sex_encoder = LabelEncoder().fit(filtered_data['Vict Sex']) vict_descent_encoder = LabelEncoder().fit(filtered_data['Vict Descent']) joblib.dump(vict_sex_encoder, 'vict_sex_encoder.pkl') joblib.dump(vict_descent_encoder, 'vict_descent_encoder.pkl') joblib.dump(model, 'crime_decision_tree.pkl') print(data.head())
